# Api Lab 1
Deployed to Vercel: https://api-lab-1.vercel.app/
## Run & build
With npm
```
npm install
npm run dev // for development
next build // for production
```
With yarn
```
yarn
yarn dev // for development
yarn build // for production
```

## Tech Stack
- https://www.weatherbit.io/api - weather REST api
- React
- TypeScript
- Tailwind CSS
- MUI
- tRPC
